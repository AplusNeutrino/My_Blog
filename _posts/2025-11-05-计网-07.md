<h2>第7章 网络安全</h2>

<h3>一、网络安全概述</h3>

<h4>1. 两类攻击</h4>

<table border="1">
<tr>
<th>类型</th>
<th>特点</th>
</tr>

<tr>
<td>被动攻击</td>
<td>窃听、不修改数据</td>
</tr>

<tr>
<td>主动攻击</td>
<td>篡改、破坏、伪造</td>
</tr>
</table>

<h4>2. 被动攻击</h4>

<ul>
  <li>截获</li>
  <li>流量分析</li>
</ul>

<h4>3. 主动攻击</h4>

<ul>
  <li>篡改</li>
  <li>恶意程序</li>
  <li>拒绝服务 DoS</li>
  <li>分布式拒绝服务 DDoS</li>
</ul>

<hr>

<h3>二、网络安全目标</h3>

<h4>CIA三要素</h4>

<table border="1">
<tr>
<th>英文</th>
<th>中文</th>
<th>含义</th>
</tr>

<tr>
<td>Confidentiality</td>
<td>保密性</td>
<td>信息不泄露</td>
</tr>

<tr>
<td>Integrity</td>
<td>完整性</td>
<td>信息不被篡改</td>
</tr>

<tr>
<td>Availability</td>
<td>可用性</td>
<td>合法用户可访问</td>
</tr>
</table>

<h4>其他安全属性</h4>

<ul>
  <li>可认证性</li>
  <li>不可否认性</li>
  <li>可控性</li>
</ul>

<hr>

<h3>三、访问控制</h3>

<h4>作用</h4>

<ul>
  <li>限制非法访问</li>
</ul>

<h4>三要素</h4>

<ul>
  <li>主体 Subject</li>
  <li>客体 Object</li>
  <li>访问控制策略</li>
</ul>

<hr>

<h3>四、数据加密模型</h3>

<h4>1. 加密基本过程</h4>

<pre>
明文 → 加密算法 → 密文
密文 → 解密算法 → 明文
</pre>

<h4>2. 密钥</h4>

<ul>
  <li>控制加密与解密</li>
</ul>

<hr>

<h3>五、密码学基础</h3>

<h4>1. 密码编码学 Cryptography</h4>

<ul>
  <li>研究加密</li>
</ul>

<h4>2. 密码分析学 Cryptanalysis</h4>

<ul>
  <li>研究破解</li>
</ul>

<h4>3. 密码体制</h4>

<ul>
  <li>算法 + 密钥</li>
</ul>

<hr>

<h3>六、对称密钥密码体制</h3>

<h4>特点</h4>

<ul>
  <li>加密密钥 = 解密密钥</li>
  <li>速度快</li>
</ul>

<h4>模型</h4>

<pre>
K加密
K解密
</pre>

<h4>常见算法</h4>

<ul>
  <li>DES</li>
  <li>3-DES</li>
  <li>AES</li>
  <li>IDEA</li>
  <li>RC4</li>
</ul>

<h4>考试高频</h4>

<ul>
  <li>DES/AES 属于对称加密</li>
</ul>

<hr>

<h3>七、非对称密钥密码体制</h3>

<h4>特点</h4>

<ul>
  <li>公钥公开</li>
  <li>私钥保密</li>
  <li>加密密钥 ≠ 解密密钥</li>
</ul>

<h4>模型</h4>

<pre>
公钥 PK 加密
私钥 SK 解密
</pre>

<h4>常见算法</h4>

<ul>
  <li>RSA</li>
  <li>ECC</li>
  <li>ElGamal</li>
</ul>

<h4>考试高频</h4>

<ul>
  <li>RSA 属于非对称加密</li>
</ul>

<hr>

<h3>八、数字签名</h3>

<h4>作用</h4>

<ul>
  <li>证明消息来源</li>
  <li>防止伪造</li>
  <li>防止抵赖</li>
</ul>

<h4>数字签名实现目标</h4>

<table border="1">
<tr>
<th>目标</th>
<th>作用</th>
</tr>

<tr>
<td>报文鉴别</td>
<td>验证发送者</td>
</tr>

<tr>
<td>报文完整性</td>
<td>验证是否被改</td>
</tr>

<tr>
<td>不可否认</td>
<td>防抵赖</td>
</tr>
</table>

<hr>

<h3>九、散列函数 Hash</h3>

<h4>特点</h4>

<ul>
  <li>任意长度 → 固定长度</li>
  <li>不可逆</li>
</ul>

<h4>作用</h4>

<ul>
  <li>消息摘要</li>
  <li>验证完整性</li>
</ul>

<h4>常见算法</h4>

<ul>
  <li>MD5</li>
  <li>SHA</li>
</ul>

<h4>考试高频</h4>

<ul>
  <li>Hash用于完整性校验</li>
</ul>

<hr>

<h3>十、IPSec</h3>

<h4>作用</h4>

<ul>
  <li>IP层安全协议</li>
</ul>

<h4>组成</h4>

<ul>
  <li>AH 认证头</li>

</ul>

<h4>1. SSL</h4>

</ul>

<h4>2. TLS</h4>

<ul>
  <li>SSL升级版</li>
</ul>
