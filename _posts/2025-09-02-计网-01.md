<h2>第1章 计算机网络导论</h2>

<h3>一、基本概念</h3>
<ul>
  <li><b>计算机网络：</b>通过通信设备和链路连接多台计算机，实现资源共享与信息传输。</li>
  <li><b>核心功能：</b>通信 + 资源共享（信息、软件、硬件）。</li>
</ul>

<h3>二、网络组成</h3>
<ul>
  <li><b>通信子网：</b>负责数据传输（链路、路由器等）。</li>
  <li><b>资源子网：</b>提供资源与服务（主机、终端、软件）。</li>
</ul>

<h3>三、网络结构</h3>
<ul>
  <li><b>边缘部分：</b>主机（用户设备）。</li>
  <li><b>核心部分：</b>路由器 + 网络（负责转发与交换）。</li>
</ul>

<h3>四、通信方式</h3>
<ul>
  <li><b>C/S：</b>客户端-服务器（典型：Web）。</li>
  <li><b>P2P：</b>对等通信（无固定服务器）。</li>
</ul>

<h3>五、接入网</h3>
<ul>
  <li>DSL、Cable、Ethernet、WiFi、3G/4G/5G</li>
  <li>区别：带宽、共享性、覆盖范围</li>
</ul>

<h3>六、网络拓扑</h3>
<ul>
  <li><b>类型：</b>星型、总线、环形、树形、网状</li>
  <li><b>考点：</b>优缺点（可靠性、成本、扩展性）</li>
</ul>

<h3>七、网络分类</h3>
<ul>
  <li><b>按范围：</b>LAN（局域网）、MAN（城域网）、WAN（广域网）、PAN</li>
  <li><b>按使用：</b>公网、专用网</li>
  <li><b>按交换：</b>电路交换、报文交换、分组交换</li>
</ul>

<h3>八、性能指标</h3>
<ul>
  <li><b>速率：</b>bit/s（bps）</li>
  <li><b>带宽：</b>最大传输能力</li>
  <li><b>吞吐量：</b>实际传输速率</li>
  <li><b>时延：</b>发送 + 传播 + 处理 + 排队</li>
</ul>

<h3>九、交换方式</h3>
<ul>
  <li><b>电路交换：</b>先建立连接，实时性好，效率低</li>
  <li><b>报文交换：</b>整包存储转发，延迟大</li>
  <li><b>分组交换：</b>分段传输（互联网核心机制）</li>
</ul>

<h3>十、分组交换补充</h3>
<ul>
  <li><b>数据报：</b>无连接，不可靠（IP）</li>
  <li><b>虚电路：</b>有连接，较可靠</li>
</ul>

<h3>十一、网络体系结构</h3>
<ul>
  <li><b>OSI七层：</b>应用、表示、会话、传输、网络、数据链路、物理</li>
  <li><b>TCP/IP四层：</b>应用层、传输层、网络层、网络接口层</li>
  <li><b>核心思想：</b>分层 + 每层提供服务</li>
</ul>

<h3>十二、协议三要素</h3>
<ul>
  <li>语法（格式）</li>
  <li>语义（含义）</li>
  <li>时序（顺序）</li>
</ul>

<h3>十三、各层关键点</h3>
<ul>
  <li><b>应用层：</b>HTTP、FTP、DNS</li>
  <li><b>传输层：</b>TCP（可靠）、UDP（不可靠）</li>
  <li><b>网络层：</b>IP、路由</li>
  <li><b>链路层：</b>帧、差错检测</li>
  <li><b>物理层：</b>比特流传输</li>
</ul>

<h3>十四、数据传输过程</h3>
<ul>
  <li>发送端：封装（逐层加头）</li>
  <li>接收端：解封装（逐层去头）</li>
</ul>

<h3>十五、发展与补充</h3>
<ul>
  <li>互联网基于 TCP/IP</li>
  <li>三网融合趋势（通信 + 广播 + 互联网）</li>
</ul>
