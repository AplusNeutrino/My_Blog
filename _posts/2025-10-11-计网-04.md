<h2>第4章 网络层</h2>

<h3>一、网络层概述</h3>

<ul>
  <li>网络层核心任务：
    <ul>
      <li>把分组从源主机传送到目标主机</li>
      <li>实现主机到主机通信</li>
    </ul>
  </li>

  <li>两大功能：
    <ul>
      <li><b>转发：</b>路由器根据转发表转发数据</li>
      <li><b>选路：</b>确定最佳路径</li>
    </ul>
  </li>

  <li>核心协议：
    <ul>
      <li>IP</li>
      <li>ARP</li>
      <li>ICMP</li>
      <li>IGMP</li>
    </ul>
  </li>
</ul>

<hr>

<h3>二、两种分组交换网络</h3>

<h4>1. 数据报网络（Internet）</h4>

<ul>
  <li>无连接</li>
  <li>每个分组独立路由</li>
  <li>可能乱序</li>
  <li>可靠性由高层保证</li>
</ul>

<h4>2. 虚电路网络 VC</h4>

<ul>
  <li>面向连接</li>
  <li>先建立连接</li>
  <li>所有分组走同一路径</li>
  <li>按序到达</li>
</ul>

<tr>

</tr>

<td>固定</td>
</tr>

<td>顺序</td>

</tr>

</tr>
</table>

<h3>三、IPv4地址</h3>

<h4>1. IPv4特点</h4>

<ul>
  <li>32位</li>
  <li>点分十进制表示</li>
  <li>例如：192.168.1.1</li>
</ul>

<h4>2. IP地址结构</h4>

<ul>
  <li>网络号 + 主机号</li>
</ul>

<h3>四、子网划分（必考）</h3>

<h4>1. 子网掩码</h4>

<ul>
  <li>作用：区分网络号与主机号</li>
  <li>网络位全1，主机位全0</li>
</ul>

<h4>2. 求网络地址</h4>

<ul>
  <li><b>网络地址 = IP地址 AND 子网掩码</b></li>
</ul>

<p>例：</p>

<ul>
  <li>IP：141.14.72.24</li>
  <li>Mask：255.255.192.0</li>
  <li>结果：141.14.64.0</li>
</ul>

<h4>3. CIDR</h4>

<ul>
  <li>无分类编址</li>
  <li>格式：IP/前缀长度</li>
  <li>例：200.23.16.0/20</li>
</ul>

<ul>
  <li>/24 → 掩码255.255.255.0</li>
</ul>

<h3>六、IP分片与重组</h3>

<ul>
  <li>原因：数据报大于MTU</li>
  <li>发送端或路由器分片</li>
  <li>目的主机重组</li>
</ul>

<h4>关键点</h4>

<ul>
  <li>除最后一个分片外：
    <ul>
      <li>数据长度必须是8字节倍数</li>
    </ul>
  </li>

  <li>MF：
    <ul>
      <li>1 = 后面还有分片</li>
      <li>0 = 最后一个分片</li>
    </ul>
  </li>

  <li>偏移量单位：
    <ul>
      <li>8字节</li>
    </ul>
  </li>
</ul>

<h4>1. 两种交付</h4>

<ul>

  <li>间接交付</li>

<ul>

    </ul>

    </ul>
  </li>
</ul>

<h3>九、ICMP协议</h3>

<h4>1. 功能</h4>

<ul>
  <li>差错报告</li>
  <li>网络诊断</li>
</ul>

<h4>2. 常见命令</h4>

<ul>
  <li>Ping：
    <ul>
      <li>测试连通性</li>
    </ul>
  </li>

  <li>Traceroute：
    <ul>
      <li>跟踪路径</li>
    </ul>
  </li>
</ul>

<h4>3. 常见差错</h4>

<ul>
  <li>目标不可达</li>
  <li>TTL超时</li>
</ul>

<hr>

<h3>十、路由协议</h3>

<h4>1. RIP</h4>

<ul>
  <li>距离向量 DV</li>
  <li>依据：跳数 hop count</li>
  <li>最大跳数：15</li>
  <li>30秒更新一次</li>
  <li>UDP 520端口</li>
</ul>

<ul>
  <li>特点：
    <ul>
      <li>简单</li>
      <li>收敛慢</li>
    </ul>
  </li>
</ul>

<h4>2. OSPF</h4>

<ul>
  <li>链路状态 LS</li>
  <li>Dijkstra算法</li>
  <li>收敛快</li>
  <li>适合大型网络</li>
</ul>

<h4>3. BGP</h4>

<ul>
  <li>边界网关协议</li>
  <li>AS之间使用</li>
  <li>运行在TCP之上</li>
  <li>TCP端口179</li>
</ul>
