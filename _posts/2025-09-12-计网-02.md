<h2>第2章 物理层 </h2>

<h3>一、物理层功能</h3>
<ul>
  <li><b>核心：</b>比特流透明传输</li>
  <li><b>4类特性：</b>
    <ul>
      <li>机械特性（接口形状/尺寸）</li>
      <li>电气特性（电压范围）</li>
      <li>功能特性（电平含义）</li>
      <li>规程特性（信号顺序）</li>
    </ul>
  </li>
</ul>

<h3>二、设备与接口</h3>
<ul>
  <li><b>DTE：</b>终端设备（计算机）</li>
  <li><b>DCE：</b>通信设备（调制解调器）</li>
  <li><b>连接方式：</b>单工 / 半双工 / 全双工</li>
</ul>

<h3>三、传输介质</h3>
<ul>
  <li><b>导向型：</b>双绞线、同轴电缆、光纤</li>
  <li><b>非导向型：</b>无线（无线电、微波、卫星）</li>
</ul>

<ul>
  <li><b>双绞线：</b>UTP / STP，最常用</li>
  <li><b>同轴电缆：</b>抗干扰强</li>
  <li><b>光纤：</b>高速、远距离、抗干扰</li>
</ul>

<h3>四、以太网标准</h3>
<ul>
  <li>10BASE-T：10Mbps，双绞线，100m</li>
  <li>100BASE-TX：100Mbps</li>
  <li><b>命名规则：</b>速率 + Base（基带）+ 介质</li>
</ul>

<h3>五、时延</h3>
<ul>
  <li><b>四类时延：</b>
    <ul>
      <li>发送时延：L / R</li>
      <li>传播时延：D / S</li>
      <li>处理时延</li>
      <li>排队时延</li>
    </ul>
  </li>
  <li><b>总时延：</b>d = d_proc + d_queue + d_trans + d_prop</li>
</ul>

<h3>六、吞吐量</h3>
<ul>
  <li>取决于链路瓶颈（最小速率）</li>
</ul>

<h3>七、数据通信基础</h3>
<ul>
  <li><b>三部分：</b>源系统、传输系统、目的系统</li>
  <li><b>信号：</b>数据的电/磁表示</li>
  <li><b>模拟 vs 数字：</b>连续 / 离散</li>
</ul>

<h3>八、通信方式</h3>
<ul>
  <li>单工（单向）</li>
  <li>半双工（轮流）</li>
  <li>全双工（同时）</li>
</ul>

<h3>九、调制与编码</h3>

<h4>1. 基带 vs 带通</h4>
<ul>
  <li>基带：直接传输数字信号</li>
  <li>带通：调制到高频载波</li>
</ul>

<h4>2. 调制方式</h4>
<ul>
  <li>AM（调幅）</li>
  <li>FM（调频）</li>
  <li>PM（调相）</li>
  <li>QAM（幅度+相位）</li>
</ul>

<h4>3. 编码方式</h4>
<ul>
  <li>非归零码</li>
  <li>曼彻斯特编码（自同步）</li>
  <li>差分曼彻斯特编码</li>
</ul>

<h4>4. PCM</h4>
<ul>
  <li>采样 → 量化 → 编码</li>
</ul>

<h3>十、奈奎斯特 & 香农（必考）</h3>
<ul>
  <li><b>奈奎斯特：</b>C = 2B log₂V（无噪声）</li>
  <li><b>香农：</b>C = B log₂(1+S/N)（有噪声）</li>
  <li><b>结论：</b>带宽↑ 或 信噪比↑ → 速率↑</li>
</ul>

<h3>十一、比特率 vs 波特率</h3>
<ul>
  <li>比特率：bit/s</li>
  <li>波特率：码元/s</li>
  <li>关系：Rb = N × log₂M</li>
</ul>

<h3>十二、同步方式</h3>
<ul>
  <li>同步通信：连续发送（效率高）</li>
  <li>异步通信：起始位+停止位（简单）</li>
</ul>

<h3>十三、复用技术</h3>
<ul>
  <li><b>FDM：</b>频分复用（不同频率）</li>
  <li><b>TDM：</b>时分复用（不同时间）</li>
  <li><b>WDM：</b>波分复用（光纤）</li>
</ul>

<h3>十四、补充</h3>
<ul>
  <li><b>CDM（码分复用）：</b>
    <ul>
      <li>每个用户使用不同编码</li>
      <li>可同时同频传输</li>
      <li>抗干扰强（典型：3G）</li>
    </ul>
  </li>

  <li><b>RS-232：</b>
    <ul>
      <li>串行通信标准</li>
      <li>DTE - DCE 接口规范</li>
      <li>短距离、低速通信</li>
    </ul>
  </li>
</ul>
