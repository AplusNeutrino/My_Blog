<h2>第8章 无线网络与移动网络</h2>

<h3>一、无线网络基础</h3>

<h4>1. WLAN（无线局域网）</h4>

<ul>
  <li>基于 IEEE 802.11 标准</li>
  <li>即 Wi-Fi</li>
</ul>

<h4>2. 两种无线局域网</h4>

<table border="1">
<tr>
<th>类型</th>
<th>特点</th>
</tr>

<tr>
<td>有固定基础设施</td>
<td>通过 AP 接入</td>
</tr>

<tr>
<td>自组网络 Ad Hoc</td>
<td>无 AP，设备直连</td>
</tr>
</table>

<hr>

<h3>二、AP（Access Point）</h3>

<h4>AP作用</h4>

<ul>
  <li>无线接入点</li>
  <li>连接无线终端与有线网络</li>
</ul>

<h4>别名</h4>

<ul>
  <li>无线接入点</li>
  <li>WAP</li>
</ul>

<h4>考试高频</h4>

<ul>
  <li>所有通信都必须经过 AP</li>
</ul>

<hr>

<h3>三、IEEE 802.11 基本概念</h3>

<h4>1. BSS（Basic Service Set）</h4>

<ul>
  <li>基本服务集</li>
  <li>无线局域网基本组成单元</li>
</ul>

<h4>2. BSA（Basic Service Area）</h4>

<ul>
  <li>BSS 覆盖区域</li>
</ul>

<h4>3. SSID</h4>

<ul>
  <li>无线网络名字</li>
  <li>最长 32 字节</li>
</ul>

<h4>4. BSSID</h4>

<ul>
  <li>AP 的 MAC 地址</li>
</ul>

<h4>考试陷阱</h4>

<table border="1">
<tr>
<th>名词</th>
<th>含义</th>
</tr>

<tr>
<td>SSID</td>
<td>无线网络名称</td>
</tr>

<tr>
<td>BSSID</td>
<td>AP 的 MAC 地址</td>
</tr>
</table>

<hr>

<h3>四、ESS（扩展服务集）</h3>

<h4>定义</h4>

<ul>
  <li>多个 BSS 通过 DS 连接形成</li>
</ul>

<h4>DS（Distribution System）</h4>

<ul>
  <li>分配系统</li>
</ul>

<h4>ESSID</h4>

<ul>
  <li>ESS 的名字</li>
</ul>

<hr>

<h3>五、无线局域网频段</h3>

<table border="1">
<tr>
<th>频段</th>
<th>用途</th>
</tr>

<tr>
<td>2.4 GHz</td>
<td>覆盖广，干扰大</td>
</tr>

<tr>
<td>5 GHz</td>
<td>速度快，干扰小</td>
</tr>
</table>

<hr>

<h3>六、802.11 标准速记</h3>

<table border="1">
<tr>
<th>标准</th>
<th>频段</th>
<th>最高速率</th>
</tr>

<tr>
<td>802.11b</td>
<td>2.4GHz</td>
<td>11 Mbps</td>
</tr>

<tr>
<td>802.11a</td>
<td>5GHz</td>
<td>54 Mbps</td>
</tr>

<tr>
<td>802.11g</td>
<td>2.4GHz</td>
<td>54 Mbps</td>
</tr>

<tr>
<td>802.11n</td>
<td>2.4GHz/5GHz</td>
<td>600 Mbps</td>
</tr>
</table>

<h4>考试高频</h4>

<ul>
  <li>b → 11 Mbps</li>
  <li>a/g → 54 Mbps</li>
  <li>n → 600 Mbps</li>
</ul>

<hr>

<h3>七、802.11 关联（Association）</h3>

<h4>作用</h4>

<ul>
  <li>移动站加入 AP</li>
</ul>

<h4>步骤</h4>

<ol>
  <li>扫描 AP</li>
  <li>选择 AP</li>
  <li>发送关联请求</li>
  <li>建立关联</li>
</ol>

<h4>两种扫描方式</h4>

<table border="1">
<tr>
<th>方式</th>
<th>特点</th>
</tr>

<tr>
<td>被动扫描</td>
<td>监听 Beacon 信标帧</td>
</tr>

<tr>
<td>主动扫描</td>
<td>主动发送探测请求</td>
</tr>
</table>

<hr>

<h3>八、802.11 MAC 层</h3>

<h4>DCF（Distributed Coordination Function）</h4>

<ul>
  <li>分布协调功能</li>
  <li>竞争方式</li>
</ul>

<h4>PCF（Point Coordination Function）</h4>

<ul>
  <li>点协调功能</li>
  <li>集中控制</li>
</ul>

<h4>考试高频</h4>

<ul>
  <li>DCF 使用 CSMA/CA</li>
</ul>

<hr>

<h3>九、CSMA/CA</h3>

<h4>全称</h4>

<ul>
  <li>载波监听多路访问/冲突避免</li>
</ul>

<h4>核心思想</h4>

<ul>
  <li>先监听再发送</li>
  <li>尽量避免冲突</li>
</ul>

<h4>与 CSMA/CD 区别</h4>

<table border="1">
<tr>
<th>协议</th>
<th>用途</th>
<th>特点</th>
</tr>

<tr>
<td>CSMA/CD</td>
<td>有线以太网</td>
<td>边发送边检测冲突</td>
</tr>

<tr>
<td>CSMA/CA</td>
<td>无线网络</td>
<td>发送前避免冲突</td>
</tr>
</table>

<hr>

<h3>十、隐藏站问题</h3>

<h4>定义</h4>

<ul>
  <li>两个站点互相听不到</li>
  <li>但都能与中间站通信</li>
</ul>

<h4>结果</h4>

<ul>
  <li>发生冲突</li>
</ul>

<h4>考试关键词</h4>

<ul>
  <li>Hidden Station Problem</li>
</ul>

<hr>

<h3>十一、暴露站问题</h3>

<h4>定义</h4>

<ul>
  <li>误以为信道忙</li>
</ul>

<h4>结果</h4>

<ul>
  <li>不必要地停止发送</li>
</ul>

<hr>
