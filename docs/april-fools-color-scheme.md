# April Fools Color Scheme

This file records the deliberately vivid light-mode test palette for future reuse.

## Source

Configured in:

```text

```

```yaml
theme_colors:
  light:
```

## Palette

| Field | Color | Element |
| --- | --- | --- |
| `background` | `#FF1744` | Page-wide background |
| `surface` | `#FF9100` | Search box, top bar, regular panel surfaces |
| `surface_alt` | `#FFD600` | Secondary surfaces and hover backgrounds |
| `card` | `#76FF03` | Post cards, status module, tag blocks |
| `border` | `#00E676` | Borders and dividers |
| `primary` | `#00E5FF` | Primary color, active states, emphasized controls |
| `accent` | `#2979FF` | Accent color, avatar ring, separators |
| `text` | `#111827` | Main text |
| `muted` | `#64748B` | Secondary text |
| `heading` | `#0F172A` | Headings |
| `sidebar_bg` | `#FF00FF` | Sidebar background |
| `sidebar_text` | `#00FF00` | Sidebar title and navigation text |
| `sidebar_muted` | `#00FFFF` | Sidebar subtitle and muted text |
| `sidebar_button_bg` | `#FFFF00` | Sidebar bottom button background |
| `sidebar_button_text` | `#0000FF` | Sidebar bottom button icons |
| `sidebar_hover_bg` | `#FF0000` | Sidebar hover and active background |
| `topbar_bg` | `#651FFF` | Top bar background |
| `link` | `#D500F9` | Links |
| `link_hover` | `#1D4ED8` | Links on hover |
| `code_bg` | `#E0EAFF` | Inline code and code block background |
| `shadow` | `rgba(255, 0, 128, 0.55)` | Card hover shadow |
